```
npm install axios
```
AJAX 쉽게 쓸수 있게 해주는 axios 설치


```
import axios from 'axios'

<button onClick={()=>{
  axios.get('https://codingapple1.github.io/shop/data2.json').then((결과)=>{
    console.log(결과.data)
  })
  .catch(()=>{
    console.log('실패함')
  })
}}>버튼</button>
```
axios 임포트 해주고 버튼 누를때 데이터 가져오게 설정을 해줌

걍 api 받아서 주소 적는데 써주면 데이터 잘 받아와짐
